# 🏏 IPL AI Analytics System

An advanced production-level AI/ML-powered cricket intelligence platform for IPL match prediction, live analytics, player insights, venue intelligence, and explainable sports analytics.

---

# 🚀 Project Overview

IPL AI Analytics System is a complete end-to-end machine learning platform designed to predict IPL match outcomes using historical data, ball-by-ball analytics, player performance, venue intelligence, toss impact, recent form, and advanced cricket statistics.

The project combines:
- Data Engineering
- Machine Learning
- Sports Analytics
- Explainable AI
- Interactive Visualization
- Real-time Analytics

to create a startup-grade cricket intelligence platform.

# 💻 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/ipl-ai-analytics.git
```

---

## Move into Project Directory

```bash
cd ipl-ai-analytics
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Streamlit Dashboard

```bash
streamlit run dashboard/app.py
```
